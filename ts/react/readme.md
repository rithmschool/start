# React TS Starter

Run `npm install` and then `npm run ...`:

| Command     | Description          |
|-------------|----------------------|
| start       | Start                |
|  build      | Build to dist dir    |
| test        | Run tests            |
| test:cov    | Run tests w/coverage |
| test:debug  | Run tests w/debugger |


### Folder Structure

No configuration or complicated folder structures, just the files you need to build your app:

```
vite-template-react
├── node_modules
├── public
│   ├── favicon.svg
└── src
    ├── App.css
    ├── App.tsx
    ├── App.test.tsx
    ├── index.css
    ├── index.tsx
    └── logo.svg
    └── setupTests.ts
├── .gitignore
├── index.html
├── package.json
├── vite.config.ts
```
